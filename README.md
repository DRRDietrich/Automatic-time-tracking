# Automatic Time Tracking

### Start Time Tracking
```bash
#!/bin/bash
nohup python3 usage.py >/dev/null 2>&1 &
```

### Show Statistics
```bash
#!/bin/bash
cat .usagelogs/YYYY_MM_DD_HH_MM_SS.txt
```
